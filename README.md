# Что это?
Тестовый стенд для модели распознания документов DeepseekOCR. С помощью утилиты make в автоматизированном режиме развертывается на
хосте и автоматически обрабатывает PNG и PDF файлы из raw_files (.png/.pdf) и сохраняет текстовое содержимое в директорию processed_files в формате Markdown (.md). 

Целью разработки тестового стенда были:
- Обкатка и тестирование предварительно собранного базового слоя контейнера longdude/transformers-base:5060ti с устанвленным стеком библиотек и инструментов для быстрой развертки ИИ на совместимых устройствах;
- Наработка навыка автоматического развертывания контейнеров с нестандартной средой выполнения;

# Предварительные требования
- Установленный Python3.11 для выполнения на хосте
- Драйвера Nvidia 575.x или новее
- Набор инструментов Nvidia CUDA 12.8 или новее 

# Технические характеристики
Тесты проводились на Nvidia GTX 1080, CUDA 6.1 (sm_61). 
Установлены CudaToolkit 1.2.9 и 1.2.1, библиотеки установлены под CUDA 1.2.1 (совместимость это весело)
Существует возможность, что с помоьщью ии был пропатчен deepseek-model (иначе почему 9 превратилось в 4.8 Гб я не знаю)
Результаты последних запусков приведены в NOTES.md (файлов логов)
