# Что это?
Подсервис системы ScInside, отвественный за получение исходного текста из PDF-статей с помощью ОКР.
Выполняется на GPU, причем скорость выполнения даже с учетом квантования ~ 80с/страницу (), но с приемлемой точностью.
Таким образом, "оценкой на глаз" максимальная скорость (если положить что в статье в среднем 20 страниц) - 1080 страниц в день или 54 статьи в среднем
